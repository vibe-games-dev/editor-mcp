# editor-mcp

Use MCP clients like Claude Code or Codex with the [vibe-games.ai](https://vibe-games.ai) browser editor.

`editor-mcp` runs locally and bridges your MCP client to the editor over a WebSocket, so tools exposed by the editor are available directly in your coding assistant.

## Setup

1. Open [vibe-games.ai](https://vibe-games.ai).
2. Go to **Settings → AI → MCP → Generate command**.
3. Choose **Claude Code** or **Codex**.
4. Paste the generated command into your shell.
5. Restart the MCP client.

The editor will connect automatically once the client starts.
