# @vibegames/editor-mcp

Use MCP clients like Claude Code or Codex with the [vibe-games.ai](https://vibe-games.ai) browser editor.

`@vibegames/editor-mcp` runs locally and bridges your MCP client to the editor over a WebSocket, so tools exposed by the editor are available directly in your coding assistant.

## Setup

1. Open [vibe-games.ai](https://vibe-games.ai).
2. Go to **Settings → AI → MCP.
3. Choose **Claude Code** or **Codex**.
4. Paste the generated command into your shell.
5. Restart the MCP client.

The editor will connect automatically once the client starts.

## MCP client config

After publishing, MCP clients can run the server through `npx`:

```json
{
  "mcpServers": {
    "vibe-games-editor": {
      "command": "npx",
      "args": ["-y", "@vibegames/editor-mcp"],
      "env": {
        "PORT": "8765",
        "EDITOR_MCP_TOKEN": "your-token"
      }
    }
  }
}
```

`PORT` and `EDITOR_MCP_TOKEN` should match the command generated by the Vibe Games editor.
